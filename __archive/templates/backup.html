Database Restoration Guide

To restore the database from a backup, follow these steps:

  1. Download the backup file you want to restore by clicking the "Download" button next to it
  2. Stop the application if it's currently running
  3. Rename the downloaded backup file to moving_boxes.db
  4. Replace the existing database file ({{ backup_config.database_path }}) with your renamed backup file
  5. Restart the application
The application will automatically use the restored database file once restarted.
